# CLI 子目录索引

这一组文档对应 `xun` 的命令入口、分发结构、典型工作流、命令速查与示例。

如果你想从“整个 CLI 是怎么组织起来的”开始理解项目，这里是最合适的入口。

## 这一组解决什么问题

- `xun` 的 CLI 入口、定义层、分发层和执行层怎么串起来
- 日常高频命令、工程辅助命令、专项能力命令分别落在哪些模块
- 常用命令应该去哪里查
- 示例和自动生成命令文档各自适合什么时候看
